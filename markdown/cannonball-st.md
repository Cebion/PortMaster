## Controls

| Button | Action |
|--|--| 
|R1|Accelerate|
|L1|Brake|
|Dpad up|Gear|
|X|Change Cam|
|Y|Pause|
|Start|Start|
|Selecti|Coin|


## Compile

```shell
Run CMake to generate the relevant build files for your compiler. You can optionally pass -DTARGET=filename to pass a custom .cmake file
```
