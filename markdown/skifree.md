## Controls

| Button | Action |
|--|--| 
|Select|Quit|
|Start|New Game|
|A|Immediate Down-right|
|B|Jump|
|X|Slight Jump|
|Y|Immediate Down-left|
|L1|Jump|
|L2|Immediate Left-face|
|R2|Immediate Right-face|
|R1|Jump|
|D-Pad|Move (as with keyboard arrow keys)|
|Left Analog|Mouse cursor for control|
|Right Analog|Mouse cursor for control|
