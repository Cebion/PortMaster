## Controls

| Button | Action |
|--|--| 
|L1|Select #1|
|L2|Select #2|
|R1|Select #4|
|R2|Select #3|
|Y|Space|
|Start + Dpad Down|Textinput|


## Compile

```shell
https://github.com/Cebion/freesynd
cd freesynd
mkdir build && cd build
cmake ..
make

```
