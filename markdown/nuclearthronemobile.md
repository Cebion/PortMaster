## Controls

|Button| Action |
|--|--|
| l-stick| movement  |
| r-stick| aim |
| R1| primary attack |
| L1| ability |
| A| select/pick up |
| B| back/cancel |
| Y/R2| change weapon |
| Start/Select| pause |
